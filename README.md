# Gesture Control (Hand Tracking)

Проект распознавания жестов руки с использованием компьютерного зрения и машинного обучения.

Система:
- отслеживает руки через камеру
- записывает примеры жестов
- обучает модель
- распознаёт жесты в реальном времени

Проект выполнен в рамках темы:

**Разработка модуля жестового управления для мультимедийных систем на основе компьютерного зрения и нейронных сетей**

## Технологии

- Python 3.10+
- OpenCV
- MediaPipe
- NumPy
- scikit-learn

## Как запустить проект

### Клонировать репозиторий

```bash
git clone https://github.com/lubowsky/gesture_control.git
cd gesture-control
```

### Создать виртуальное окружение
```bash
python -m venv .venv
.venv\Scripts\activate
```

### Установить зависимости
```bash
pip install -r requirements.txt
```

### Запустить проект
```bash
python main.py
```

## Обучение жестов

Перед использованием системы необходимо записать примеры жестов.

Запустить программу:
```bash
python main.py
```

Во время работы программы нужно показать жест в камеру и нажать соответствующую клавишу.

Клавиша	Жест	Значение
1	help	помогите
2	open	откройте
3	drink	пить воду
4	eat	    кушать

После нажатия клавиши программа записывает последовательность движения руки и сохраняет данные в папку: dataset

Рекомендуется записать: 30–100 примеров каждого жеста

## Обучение модели

После записи жестов необходимо обучить модель.

Запустить:
```bash
python train_model.py
```

После успешного обучения появится файл: gesture_model.pkl
Это обученная модель распознавания жестов.

## Запуск распознавания жестов

После обучения модели снова запустить программу:
```bash
python main.py
```

Программа:
- включает веб-камеру
- отслеживает руки
- анализирует движение
- выводит распознанный жест на экран

## Описание жестов
Жесты обучаются пользователем, поэтому возможны небольшие различия в исполнении.

- Help (помогите)
Открытые ладони направлены вверх.

- Open (откройте)
Жест раскрытия руками.

- Drink (пить воду)
Имитация поднесения руки к рту, как будто пьёте воду.

- Eat (кушать)
Имитация движения руки ко рту.

## Управление программой

Клавиши управления:
ESC — выход из программы
1 — запись жеста help
2 — запись жеста open
3 — запись жеста drink
4 — запись жеста eat

###  Принцип работы системы

## Алгоритм работы программы:

1. Камера получает изображение.
2. MediaPipe определяет положение руки.
3. Выделяются 21 ключевая точка руки.
4. Формируется последовательность координат.
5. Модель машинного обучения анализирует движение.
6. Определяется жест пользователя.
7. Результат выводится на экран.
